VERBA v1.0 — THE NATURAL LANGUAGE

The Language of Humans.

Verba is an interpreted, English-like programming language designed to be readable by anyone, even without a programming background.

hello.vrb 
/- A clean welcome
ask the user "What is your name?" and save to name.
say "Hello, {name}! Welcome to Verba.".
100% Readable
0.1ms Fast Execution
20+ Core Modules

Installation

Verba requires Python 3.10 or newer.

# Install locally
pip install -e .

# Run a script
verba run project.vrb

CLI Commands

The verba tool provides everything you need to manage your lifecycle.

CommandEffect
verba run <file.vrb>Execute a Verba script.
verba watch <file.vrb>Watch script for changes and re-run automatically.
verba check <file.vrb>Parse only — syntax validation without running.

verba format <file.vrb>Auto-format with 4-space indentation.

verba update <pkg>Update an installed package (or all packages in verba.json).
verba remove <pkg>Remove an installed package from modules/ and update verba.json.

General Syntax

Verba is designed to feel like instructions to a human. Every statement ends with .; blocks open with : and close with end.

/- single-line comment
# also a comment
/-- block comment --/
say "Every statement ends with a period".
if true:
    say "Blocks open with colon, close with end.".
end.

Constants & Immutability

Verba supports constants using the constant keyword. Once a constant is assigned, it cannot be modified or reassigned.

constant PI = 3.14.
PI = 3.15.  /- Error: Cannot re-assign to constant 'pi'

constant APP_NAME = "Verba Language".
APP_NAME += " (v2)". /- Error: Cannot re-assign to constant 'app_name'

To allow redefining a constant name, you must first explicitly free it:

constant X = 10.
free X.
constant X = 20. /- Success

Null Safety & Option Types

Use some value when a value exists and none when it does not. This gives you a safer alternative to raw null for optional data.

name = some "Verba".
missing = none.

if name is some:
    safe_name = the result of running name.unwrap.
    say "Resolved: ", safe_name.
end.

fallback = the result of running missing.unwrap_or with "Anonymous".
say fallback.

if missing is none:
    say "No value available.".
end.
FeatureMeaning
some valueCreate an Option containing a value.
noneCreate an empty Option.
is some / is noneCheck whether an Option holds a value.
option.unwrapGet the value or raise an error if empty.
option.unwrap_or with xGet the value or use a fallback.
option.or_else with xReturn the original Option or a fallback Option.
option.map with fnApply function to inner value.
option.filter with predFilter inner value by predicate.
option.and_then with fnChain another Option-returning function.

Notes & Docstrings

note writes human-readable inline descriptions. The first note in a define or class body becomes its official docstring shown by help.

note This module computes geometry.

define area needing w, h:
    note Returns width times height.
    give w * h.
end.

Help System

The built-in help command prints documentation at runtime — in scripts or the REPL.

help.                    /- list all modules
help strings.            /- list strings module functions
help strings.upper.      /- show params for strings.upper
help myFunction.         /- show your function's docstring
help MyClass.            /- show class methods

Conditions

    say "Big".
else:
    say "Small".

Checks such as is null, is not null, is some, and is none are all supported in conditions.

Loops

end.

for i from 1 to 10:
    say i.
end.

for i from 0 to 100 step 10:
    say i.
end.

Pattern Matching

Verba features a deep pattern matcher supporting literal values, wildcards, and type checks. COLONS are required at the end of each branch, but trailing periods are optional for concise code.

match val:
    when 1: say "Exactly one".
    when "Verba": say "A word match".
    
    /- Type patterns
    when is number: say "Is a numeric value".
    when is word:   say "Is a string".
    when is flag:   say "Is a boolean".
    when is some:   say "Contains a value".

    /- Structure patterns
    when {"type": "human", "name": _}: say "A person with any name".
    when [1, _, 3]: say "List starting with 1 and ending with 3".

    /- Universal wildcard
    when _: say "Matches anything!".
    
    else: say "Fallback match".
end.

Supported types for is: number, word, flag, list, map, none, some.

With Statement

Binds an expression to a name for the duration of a block. If the value has a close method, it is called automatically on exit.

with the result of running db.open with "data.db" as conn:
    rows = the result of running conn.query with "SELECT * FROM users".
    say (length of rows).
end.
/- conn.close called automatically

Parallelism (Multiprocessing)

While async runs code in micro-threads within the same OS process, parallel spawns separate OS processes to leverage multiple CPU cores. Communication is handled via return values.

define calculate_pi needing precision:
    /- Heavy CPU work here...
    give 3.14159.
end.

/- Spawn parallel task
parallel run calculate_pi with 1000000 and save to task_p.

/- Main process continues...
say "Task started in another process!".

/- Wait and retrieve result
join result = task_p.
say "Final Pi: ", result.

Tip: Use parallel for CPU-intensive tasks and async for I/O-bound tasks.

Decorators

Apply @decorator_name directly above a define block. Multiple decorators stack.

DecoratorEffect

@timePrints execution time after each call.

@time

Canvas — 2D Drawing & Animation

Built on tkinter. No extra dependencies. Supports shapes, text, images, events, and animation loops.

/- create an 800×600 window
c = the result of running canvas.new with "My App", 800, 600.
run c.background with "#1e1e2e".

/- shapes
run c.rect   with 50, 50, 200, 100, fill = "#a6e3a1".
run c.circle with 400, 300, 60,  fill = "#cba6f7", outline = "#b4befe".
run c.line   with 0, 0, 800, 600, color = "#f38ba8", width = 2.
run c.text   with 10, 10, "Hello!", color = "#cdd6f4", size = 20.

/- click event — sets canvas_x, canvas_y globals
run c.on_click with "on_click".
define on_click:
    run c.circle with canvas_x, canvas_y, 15, fill = "#f9e2af".
end.

/- 60 FPS animation loop
bx = 400. vx = 4.
define frame:
    run c.clear.
    bx += vx.
    if bx > 780: vx = 0 - vx. end.
    if bx < 20:  vx = 0 - vx. end.
    run c.circle with bx, 300, 30, fill = "#cba6f7".
end.
run c.loop with "frame", 60.
